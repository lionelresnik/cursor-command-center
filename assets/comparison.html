üì¶ Repomix Workflow

  1. Run npx repomix@latest
  2. Get repomix-output.xml
  3. Copy file contents
  4. Paste into ChatGPT/Claude
  5. Get code suggestions
  6. Copy code back to files
  7. Code changed? ‚Üí Step 1 again

üéØ Command Center Workflow

  1. Run ./open.sh (once)

  5. AI edits files directly
  6. Code changed? ‚Üí Already indexed!
